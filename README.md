## 📖 About   

**QuickFix** is a front-end web project that simulates an on-demand vehicle repair booking platform. It allows users to request a mechanic to their doorstep, track their request status, and view available services — all within a clean, single-page HTML interface.

Whether it's a dead battery, flat tyre, or a check engine light, QuickFix connects drivers with vetted, nearby mechanics quickly and transparently.

---

## ✨ Features

- 🚗 **Service Listings** — Browse popular services with upfront pricing (Battery, Tyre, Oil Change, Diagnostics & more)
- 📋 **Booking Form** — Request a mechanic with vehicle details, preferred time, and location
- 📍 **Live Location Support** — "Use my location" button for quick address input
- 📦 **Request Tracker** — Track active booking status in real time (UI)
- 📱 **Responsive Design** — Mobile-friendly layout with a hamburger nav menu
- ⭐ **Testimonials** — Real-feel customer reviews section
- 🛡️ **7-Day Warranty Badge** — Builds trust with service guarantee callouts

## 🚀 Getting Started

To run this project locally:

```bash
# 1. Clone the repository
git clone https://github.com/subhupanda21/QuickFix.git

# 2. Navigate into the project folder
cd QuickFix

# 3. Open in your browser
open index.html
```

Or simply double-click `index.html` to open it in any browser.
